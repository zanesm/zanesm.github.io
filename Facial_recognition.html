About This Project:

    • About: A two part capstone project created while working at Five Star. Both portions of the project were used to effectively showcase how my Computer Science degree has equipped me with the tools and knowledge to design and implement innovative solutions to real-world problems.

      PART ONE: The first portion involved implementing and configuring a PoE (Power over Ethernet) security camera system. This part was more IT-focused and included planning, hardware installation, network configuration, and system testing. The goal was to design and deploy a modern, secure camera network that improved monitoring and integrated smoothly with the school district's existing IT infrastructure.

      PART TWO: The second part involved designing and developing a lightweight facial recognition system using a Raspberry Pi and a camera module. The system leveraged Python and OpenCV for facial detection and recognition. I also built a GUI that allows users to register or remove individuals from the database. Additional functionality was added to send alerts when an unknown face is detected and to log attendance automatically.

Lessons Learned:

  • Gained hands-on experience with real-world hardware and networking, including PoE setup, VLAN configuration, and MAC-based authentication.
  • Improved problem-solving skills through troubleshooting hardware issues, IP conflicts, and mislabeled cabling in a live environment.
  • Developed a practical understanding of deploying security-focused camera systems that integrate into existing IT infrastructure.
  • Enhanced Python development skills, particularly in image processing, GUI design, and modular coding for facial recognition systems.
  • Learned to implement machine learning concepts using OpenCV and face-recognition libraries in a constrained embedded environment (Raspberry Pi).
  • Strengthened UI/UX design skills by building an intuitive GUI that could be used by non-technical users for facial recognition tasks.
  • Applied time-delay logic and real-time video processing to solve real-world alerting and system noise challenges.
  • Recognized and addressed limitations in scalability and facial recognition accuracy, especially with user name collisions and larger datasets.
  • Connected classroom learning in networking, biometrics, HCI, and IoT to tangible deliverables with measurable security impact.
