About This Project:

This project allows users to search for NBA player statistics by name and season, using data pulled from the balldontlie API. Users input a playerâ€™s name and season year, and the app returns their season averages, position, team, and other career details, along with an embedded YouTube highlight video.

Built with:

  • HTML & CSS (Frontend)
  • Python (Backend via Bottle Web Framework)
  • API Integration with balldontlie
  • Dynamic YouTube Video Embedding

Key Features:

  • Form input for player name and season year
  • Displays player height, weight, position, and team info
  • Shows per-game season averages (PTS, REB, AST, FG%, etc.)
  • Auto-embeds a YouTube clip of the searched player

Example Search

Searching LeBron James for the 2023 season will display:

  • Name: LeBron James
  • Team: Los Angeles Lakers
  • Games Played: 55
  • Points per Game: 25.4
  • Assists: 6.8, Rebounds: 7.2
  • Highlight video of LeBron from 2023 season
