About This Project:

  • About: This project was designed to replicate the functions of Twitter. Upon starting the application, the GUI prompts users to log in. If they do not have an account, they are then able to create one. Once logged in, users are able find others to connect with and follow them, by searching for a specific user, or viewing all users on the platform. Furthermore, users are able to make posts, or edit posts they have already made. Additional features allow for users to like a post, or look at a "feed" view that displays new posts from people they follow. The program also features an admin menu, allowing for the admin to delete users or posts, or edit user information, such as passwords. The program was written in Java during my second semester with IUPUI. User data is stored in an SQL database.
