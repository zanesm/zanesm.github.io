About This Project:

  • About: This project smart door lock created using an Arduino Uno R4, RFID scanner, RFID badge, servo motor, and a slide bolt latch. Code for the project was written in C++. The project begins with the LCD screen prompting the user to scan their RFID badge for authentication. The user then scans the badge, and if they have valid credentials, this will trigger the servo motor to rotate, which slides the bolt on the lock over. The door will stay unlocked for several seconds, then automatically relock. If the user has invalid credentials, the LCD screen will output a message notifying the user that access has been denied, and the servo motor will do nothing. Furthermore, the user is able to access the controls of the lock via a web page. he web page allows the user to unlock the door remotely. After unlocking, the system automatically relocks the door after a few seconds
