About This Project:

  • About: A simple Hangman game using HTML, JavaScript, and the p5.js library. The HTML part sets up the basic structure of the game, including a title, buttons for each letter of the alphabet, and placeholders for the hangman drawing, guessed letters, and the answer slots. The JavaScript part handles the game logic, including selecting a random word from a predefined list, tracking correct and incorrect guesses, updating the display based on user input, styling, and drawing the hangman or win/lose screens using p5.js.
  • Course: INTRO TO MEDIA APPLICATION DEVELOPMENT
  • Project Description: "This final project should employ a majority of the skills you learned in this course to produce a working application that responds to several, temporal interactions from a user, modifies a program state in response, and produces a visual change to the change in the application state. Applications will be graded based on: technical mastery, completion, commenting, and process."
  • Project Requirements: "Your program must include at least 1/2 of the following options: mouse interaction, 2 global variables access and modified in response to user input, an array that is accessed or modified in response to user interaction, a if/else if/else block, a loop, an object that is modified during application runtime, manipulating the document object."
  • Grade: 20/20

Lessons Learned:

  • How to use the p5.js library to create interactive graphics and animations in a web page.
  • How to structure a simple game using HTML, CSS, and JavaScript.
  • How to handle user input and update the display based on the game state.
  • How to use conditional statements and loops to control the flow of the game.
  • How to create a responsive and visually appealing user interface using CSS.
